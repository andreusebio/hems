Server was built on a Raspberry Pi 3B and uses multiple softwares and tools:

-NodeRED: its comes pre-installed on Raspbian operating system, but should be updated

-Mosquitto (MQTT broker): mosquito can be installed using the following commad:
sudo apt-get install mosquito
tutorial: https://youtu.be/AsDHEDbyLfg?t=1m17s

-OpenVPN (VPN server): http://www.pivpn.io/#install
Convenient tutorial: https://youtu.be/BwLBq1j1sws?t=4m54s

-SQLite (database)
Tutorial: https://youtu.be/OFi4qsoy0CM?t=1m25s

-phpLiteAdmin (tool to manage database) not completely necessary but convenient
Tutorial: https://randomnerdtutorials.com/how-to-install-phpliteadmin-database-in-a-raspberry-pi/
